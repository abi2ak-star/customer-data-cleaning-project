# Customer Data Cleaning with Python & Pandas

This project demonstrates how messy customer data was cleaned using Python and Pandas.

## Tasks completed

- Removed duplicate records
- Handled missing values
- Converted invalid data types
- Generated summary statistics
- Exported cleaned dataset

## Files

messy_customer_data.xlsx → raw dataset

clean_customer_data.xlsx → cleaned dataset

customer_data_cleaning.ipynb → full cleaning workflow
